Detailed Description

template<typename _KeyType, typename _RequestType, typename _ItemType, bool _CompressWAL>
class jelly::Node< _KeyType, _RequestType, _ItemType, _CompressWAL >

Base class for LockNode and BlobNode.

Contains a bunch of shared functionality as they are conceptually very similar. Applications should not use this class template directly.
