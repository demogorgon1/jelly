Detailed Description

template<typename _KeyType, typename _ItemType>
class jelly::Backup< _KeyType, _ItemType >

Holds the state of a backup job.

In order to do a backup, first call Node::StartBackup() on main thread, then call Backup::Perform() on any thread, and finally call Node::FinalizeBackup() on the main thread.
