Detailed Description

template<typename _KeyType, typename _MetaType = MetaData::Dummy, bool _AutoIncrementSeq = false>
class jelly::BlobNode< _KeyType, _MetaType, _AutoIncrementSeq >

A node for storing blobs.

Template Parameters
_KeyTypeKey type. For example UIntKey.
_MetaTypeMeta data type.
_AutoIncrementSeqAutomatically increment blob sequence numbers when updated.
Examples
Example1_BlobNode.cpp, Example3_Housekeeping.cpp, and Example4_HousekeepingAdvisor.cpp.
